# Layout 布局

布局组件，快速搭建结构。

## 基础使用

使用 `vi-row` 来定义行，`vi-col` 来定义列。

<vi-row>
  <vi-col>1</vi-col>
  <vi-col>2</vi-col>
  <vi-col>3</vi-col>
</vi-row>

<vi-row>
  <vi-col>1</vi-col>
  <vi-col>2</vi-col>
  <vi-col>3</vi-col>
</vi-row>

```vue
<template>
  <vi-row>
    <vi-col>1</vi-col>
    <vi-col>2</vi-col>
    <vi-col>3</vi-col>
  </vi-row>
</template>
```
